# Aliases, Completions, Common Dir & Config Path Change

## Overview

Three changes:
1. Move config path from `~/.config/ctx/` to `~/Library/ctx/` (macOS standard)
2. Add `~/projects/common/` directory with global shell aliases and completions
3. Add `ctx aliases generate` and `ctx completions generate` commands
4. Update `ctx shell-init zsh` to source aliases and completions

## Config Path Change

Default config directory changes from `~/.config/ctx/` to `~/Library/ctx/`.

Files affected:
- `src/ctx/config.py` — `get_config_dir()` default
- `src/ctx/shell.py` — hardcoded paths in shell hook
- `src/ctx/skills.py` — `CLAUDE_MD_CONTENT` reference
- All 12 skills — Company Context section
- `templates/projects-claude.md`
- `README.md`, `CLAUDE.md`

`CTX_CONFIG_DIR` env var override still works.

## Completions (`~/projects/common/completions.sh`)

Generated by `ctx completions generate`. Loads native tool completions and maps them to aliases.

```bash
# Native completions
source <(kubectl completion zsh 2>/dev/null)
source <(helm completion zsh 2>/dev/null)
source <(gh completion -s zsh 2>/dev/null)
source <(glab completion -s zsh 2>/dev/null)
source <(docker completion zsh 2>/dev/null)

# Alias completions
compdef k=kubectl
compdef h=helm
compdef tf=tofu
compdef tg=terragrunt
compdef ap=ansible-playbook
compdef ns=nomad
compdef nj=nomad
compdef vs=vault
compdef cm=consul
compdef d=docker
compdef dc='docker compose'
compdef g=git
```

## Shell Integration Update

`ctx shell-init zsh` now outputs:

```bash
# Common aliases and completions
[[ -f ~/projects/common/aliases.sh ]] && source ~/projects/common/aliases.sh
[[ -f ~/projects/common/completions.sh ]] && source ~/projects/common/completions.sh

# ctx env and prompt (existing hook)
_ctx_precmd() {
  local env_file="${CTX_CONFIG_DIR:-$HOME/Library/ctx}/state.env"
  ...
}
```

## New Commands

```
ctx aliases generate       # generate ~/projects/common/aliases.sh
ctx completions generate   # generate ~/projects/common/completions.sh
```

Both are idempotent. They overwrite existing files.
